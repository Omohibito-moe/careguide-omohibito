# もしもナビ（オリジナル実装）

このディレクトリには、元の「もしもナビ」のコードと資料を保管します。

## ディレクトリ構成

```
moshimo-navi/
├── README.md                  （このファイル）
├── original-code/             （元のもしもナビのソースコード）
│   └── .gitkeep
└── slides/                    （スライド画像などの資料）
    └── .gitkeep
```

## A〜E構造

もしもナビのコア構造（スライドより）：

- **A: 医師・看護師の話を聞く**
  - 病状・回復見込み
  - 必要な生活支援や医療行為は何か

- **B: 家族で話し合う**
  - 自宅で過ごすか施設に入るか
  - どんな生活を送りたいか

- **C: 自宅の場合 介護体制構築**
  - 介護保険の申請
  - 住宅改修
  - 福祉用具

- **D: 施設の場合 施設探し**
  - 立地・コストなど

- **E: 仕事・職場との調整をする**
  - 介護休業や時短勤務など

## 参照目的

現在のケアガイドに、このA〜E構造を統合するための参考資料として使用します。
